Kaazing WebSocket Gateway (Community Edition)

A high-performance, scalable, secure, and robust real-time platform for reactive event-driven web and mobile applications.

Release notes:   https://github.com/kaazing/gateway/releases
Documentation:   https://kaazing.com/doc/5.0/
Getting started: https://kaazing.com/getting-started/

To start the gateway:

    Windows:   % bin\gateway.start.bat
    Linux/Mac: $ bin/gateway.start

Once started, use a browser and open http://localhost:8000

See https://kaazing.com/download/#ee-kwg for more information on starting the gateway.
